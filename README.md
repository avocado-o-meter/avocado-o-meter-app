# 🥑 avocado-o-meter - v1.0.0

## Tech Stack
- 🔥 React Native CLI
- 🏗️ React Native TypeScript Template
- 🛂 Strict TypeScript
- 🚨 ESLint & Type-Check
- 🍖 Axios

## How to use

Install dependencies:

```bash
yarn install
```
### Run App

#### Run from MacOs Terminal.

➡️ Android:

```bash
npx react-native run-android
```
